Smart Mobility Robot for Autonomous Object Transportation in Factory Warehouses

Project Detail Page

Abstract

The Smart Mobility Robot is designed to automate object transportation in factory warehouses using a Raspberry Pi 5 based control system. The robot is driven by planetary gear motors connected to a motor driver, which provide smooth differential drive movement for navigation. The system operates in two modes: Autonomous Mode and Remote Control Mode, allowing flexible operation depending on the task. In autonomous mode, a camera module scans QR codes to identify locations such as Room 1 and Room 2, enabling the robot to navigate automatically along predefined routes. An OLED display shows the current operating mode, robot location, and movement status, while a touch sensor is used to switch between modes. In remote control mode, the robot can be manually operated using a FlySky transmitter with an FS-iA6B receiver, making the system suitable for efficient and intelligent warehouse automation.

System Working Principle

Mode 1: Autonomous Navigation Mode

  1. System powers on and initializes modules.
  2. OLED shows current status and mode.
  3. Camera scans QR code for destination (Room 1 / Room 2).
  4. Raspberry Pi processes destination route map.
  5. Robot navigates using differential motor control.
  6. OLED shows mode, destination, and status.
  7. After delivery, robot returns to start position.

Mode 2: Remote Control Mode

  • Touch sensor (TTP223) switches Autonomous and Remote mode.
  • Operator uses FlySky FS-i6 transmitter.
  • FS-iA6B receiver sends control signal to Raspberry Pi.
  • Robot moves manually based on operator commands.
